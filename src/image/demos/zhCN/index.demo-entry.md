## Props

### Image Props

| 名称 | 类型 | 默认值 | 说明 |
| --- | --- | --- | --- |
| src | `string` | `undefined` | 图片来源 |
| alt | `string` | `undefined` | 图片说明 |
| width | `string \| number` | `undefined` | 图片宽度 |
| height | `string \| number` | `undefined` | 图片高度 |
| show-toolbar | `boolean` | `true` | 图片放大后是否展示底部工具栏 |

### ImageGroup Props

| 名称         | 类型      | 默认值 | 说明                         |
| ------------ | --------- | ------ | ---------------------------- |
| show-toolbar | `boolean` | `true` | 图片放大后是否展示底部工具栏 |
