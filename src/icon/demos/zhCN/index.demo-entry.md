# 图标 Icon

Naive-UI 推荐使用 <n-a href="https://vicons.vercel.app/">vicons</n-a> 作为图标库。

## 演示

```demo
basic
theme
custom-icon
depth
```

## Props

| 名称  | 类型                          | 默认值      | 说明 |
| ----- | ----------------------------- | ----------- | ---- |
| color | `string`                      | `undefined` |      |
| depth | `1 \| 2 \| 3 \| 4 \| 5`       | `undefined` |      |
| size  | `number \| string`            | `undefined` |      |
| theme | `'light' \| 'dark' \| string` | `undefined` |      |
