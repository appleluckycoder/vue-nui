## Props

| 名称            | 类型                             | 默认值      | 说明 |
| --------------- | -------------------------------- | ----------- | ---- |
| default-value   | `boolean`                        | `false`     |      |
| disabled        | `boolean`                        | `false`     |      |
| size            | `'small' \| 'medium' \| 'large'` | `'medium'`  |      |
| value           | `boolean`                        | `undefined` |      |
| on-update:value | `(value: boolean) => void`       | `undefined` |      |
