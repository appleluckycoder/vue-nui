# Carousel 轮播图

一般用来播放一些好消息。

## 演示

```demo
basic
autoplay
```

## API

### Carousel Props

| 名称     | 类型      | 默认值  | 说明           |
| -------- | --------- | ------- | -------------- |
| autoplay | `boolean` | `false` | 是否自动播放   |
| interval | `number`  | `5000`  | 自动播放的间隔 |

### Carousel Slots

| 名称    | 参数 | 说明       |
| ------- | ---- | ---------- |
| default | `()` | 轮播的内容 |
