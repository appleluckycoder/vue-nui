## API

### Carousel Props

| Name     | Type      | Default | Description                      |
| -------- | --------- | ------- | -------------------------------- |
| autoplay | `boolean` | `false` | Whether to scroll automatically. |
| interval | `number`  | `5000`  | Auto play interval.              |

### Carousel Slots

| Name    | Parameters | Description       |
| ------- | ---------- | ----------------- |
| default | `()`       | Carousel content. |
