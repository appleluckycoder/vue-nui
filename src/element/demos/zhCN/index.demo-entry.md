# 元素 Element

Element 上面有很多 Naive UI 提供的主题变量。

## 演示

```demo
basic
color
```

## Props

| 名称     | 类型      | 默认值  | 说明                           |
| -------- | --------- | ------- | ------------------------------ |
| abstract | `boolean` | `false` | 是否渲染一个 Wrapper DOM       |
| tag      | `string`  | `'div'` | `n-element` 需要渲染为什么元素 |

## Slots

| 名称    | 参数                           | 说明 |
| ------- | ------------------------------ | ---- |
| default | `(themeVars: CommonThemeVars)` |      |
