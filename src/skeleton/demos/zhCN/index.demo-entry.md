## Props

| 名称     | 类型                             | 默认值      | 说明 |
| -------- | -------------------------------- | ----------- | ---- |
| text     | `boolean`                        | `false`     |      |
| round    | `boolean`                        | `false`     |      |
| circle   | `boolean`                        | `false`     |      |
| height   | `string \| number`               | `undefined` |      |
| width    | `string \| number`               | `undefined` |      |
| size     | `'small' \| 'medium' \| 'large'` | `undefined` |      |
| repeat   | `string \| number`               | `1`         |      |
| animated | `boolean`                        | `true`      |      |
| sharp    | `boolean`                        | `true`      |      |
