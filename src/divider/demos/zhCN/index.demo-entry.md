# 分割线 Divider

分割一些内容。

## 演示

```demo
basic
content
vertical
```

## Props

| 名称            | 类型                            | 默认值     | 说明 |
| --------------- | ------------------------------- | ---------- | ---- |
| dashed          | `boolean`                       | `false`    |      |
| title-placement | `'left' \| 'right' \| 'center'` | `'center'` |      |
| vertical        | `boolean`                       | `false`    |      |

## Slots

| 名称    | 参数 | 说明         |
| ------- | ---- | ------------ |
| default | `()` | 分割线的标题 |
