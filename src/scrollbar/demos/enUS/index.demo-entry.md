## API

### Scrollbar Props

| Name         | Type      | Default | Description                         |
| ------------ | --------- | ------- | ----------------------------------- |
| x-scrollable | `boolean` | `false` | Whether it can scroll horizontally. |

### Scrollbar Slots

| Name    | Parameters | Description     |
| ------- | ---------- | --------------- |
| default | `()`       | Scroll content. |
