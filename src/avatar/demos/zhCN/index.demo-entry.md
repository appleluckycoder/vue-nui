# 头像 Avatar

在互联网上，没有人知道你是一条狗。

## 演示

```demo
size
shape
color
badge
icon
name-size
```

## Props

| 名称 | 类型 | 默认值 | 说明 |
| --- | --- | --- | --- |
| size | `'small' \| 'medium' \| 'large' \| number` | `'medium'` | 头像的尺寸 |
| src | `string` | `undefined` | 头像的地址 |
| round | `boolean` | `false` | 头像是否圆形 |

## Slots

| 名称    | 参数 | 说明             |
| ------- | ---- | ---------------- |
| default | `()` | 头像内填充的内容 |
