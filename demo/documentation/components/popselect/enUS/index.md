# Popselect

If you want select some options but don't want a picker, you can use popselect instead.

## Demos
```demo
basic
cancelable
custom-width
multiple
```

## Props

|name|type|default|description|
|-|-|-|-|
|value|`string \| number`|||
|options|`Array`|||
|width|`number`|||
|multiple|`boolean`|||
|cancelable|`boolean`|||
|controller|`Object`|||
|arrow|`boolean`|||
|trigger|`'click' \| 'hover' \| 'manual'`|||
