# 元素 Element
Element 可以被渲染成各种标签同事还可以获得某些 Config Provider 提供的配置。

## 演示
```demo
basic
color
```

## Props
|名称|类型|默认值|说明|
|-|-|-|-|
|as|`string`|`'div'`||

## Slots
|名称|参数|说明|
|-|-|-|
|default|`(theme: string, namespace: string, themeEnvironment: any, styleScheme: object)`||

## Events
|名称|参数|
|-|-|
|theme-change|`(theme: string)`|
