# Size
Card has `small`, `medium`, `large`, `huge` sizes.
```html
<n-card title="Small Card" size="small">
  Card Content
</n-card>
<n-card title="Medium Card" size="medium">
  Card Content
</n-card>
<n-card title="Large Card" size="large">
  Card Content
</n-card>
<n-card title="Huge Card" size="huge">
  Card Content
</n-card>
```
```css
.n-card {
  margin-bottom: 12px;
}
```
