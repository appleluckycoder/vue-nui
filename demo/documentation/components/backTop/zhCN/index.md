## Props
|名称|类型|默认值|说明|
|-|-|-|-|
|target|`() => HTMLElement`|一个返回最邻近可滚动父级元素的函数||
|right|`number`|`40`||
|bottom|`number`|`40`||
|visibility-height|`number`|`180`||
|themed-style|`object`|`null`||
