# Color

Color is depend on you even in different theme. You can set it to something related to things you want to eat.

```html
<n-avatar
  :themed-style="{
  light: {
    color: 'yellow',
    backgroundColor: 'red'
  },
  dark: {
     color: 'red',
    backgroundColor: 'yellow'
  }
}"
  >M</n-avatar
>
```
