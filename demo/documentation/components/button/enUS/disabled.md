# Disabled
Button can be disabled
```html
<n-button
  size="small"
  disabled
>
  Disabled
</n-button>
<n-button
  disabled
  type="primary"
>
  Disabled
</n-button>
<n-button
  disabled
  round
  type="primary"
>
  regular size
</n-button>
<n-button
  disabled
  circle
  icon="md-save"
/>
```
```css
.n-button {
  margin: 0 8px 8px 0;
}
```
