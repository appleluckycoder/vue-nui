# 表格 Table
<!--single-column-->
如果你只想画点简单的表格，用它。需要渲染数据请看<n-a to="n-data-table">数据表格 Data Table</n-a>。

## 演示
```demo
basic
bordered
single-column
single-line
```

## Components
你可以使用 `n-table`、`n-thead`、`n-tbody`、`n-tr`、`n-th` 和 `n-td`。

## Props
### Table Props
|名称|类型|默认值|说明|
|-|-|-|-|
|bordered|`boolean`|`true`||
|single-line|`boolean`|`true`||
|single-column|`boolean`|`false`||
